一、date指令

date指令基本语法:

1. date    （功能描述：显示当前时间）      

2. date +%Y    （功能描述：显示当前年份）      

3. date +%m    （功能描述：显示当前月份）      

4. date +%d    （功能描述：显示当前是哪一天）

5. date "+%Y-%m-%d %H:%M:%S"   （功能描述：显示年月日时分秒）
6. date  -s  字符串时间 （设置系统当前时间）

```shell
date -s "2019-11-11 11:11:11" #设置系统当前时间为2019-11-11 11:11:11
```

二、cal指令

cal指令基本语法:

1. cal [选项] （不加选项为本月指令）

2. cal 年份 （显示该年日历）

```shell
cal 2019 #查看2019年日历
```
